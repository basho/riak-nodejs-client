FetchBucketProps Class

Extends CommandBase
Defined in: lib/commands/kv/fetchbucketprops.js:27
Module: KV

Command used to fetch a bucket's properties from Riak.

As a convenience, a builder class is provided:

var fetch = new FetchBucketProps.Builder()
    .withBucketType('my_type')
    .withBucket('myBucket')
    .withCallback(myCallback)
    .build();

See FetchBucketProps.Builder

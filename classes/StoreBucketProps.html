StoreBucketProps Class

Extends CommandBase
Defined in: lib/commands/kv/storebucketprops.js:30
Module: KV

Command used to set the properties on a bucket in Riak.

As a convenience, a builder class is provided:

var storeProps = new StoreBucketProps.Builder()
             .withAllowMult(true)
             .build();

See StoreBucketProps.Builder
